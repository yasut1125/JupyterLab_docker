## 自分用のJupyterLabのテンプレ

docker_xxx/ (optional directory name) <br>
├─src/ (work directory)<br>
├─Dockerfile <br>
├─docker-compose.yml <br>
└─requirements.txt <br>


## Start
run containers
```bash
docker compose run jupyter
```
stop and remove containers
```
docker compose down
```
start containers
```
docker compose up
```
